DICOM Viewer

Anzeige

Damit sich Bilder im DICOM Format anzeigen lassen, werden DICOM Viewer vorausgesetzt. Mit diesen lassen sich nebst dem betrachten der Bilder, auch viele verschiedene Veränderungen am Bild vornehemen. Dieses Kapitel wird mit dem Programm DICOM Sante PACS behandelt, welches über einen übersichtlichen Viewer verfügt. Er bietet zudem auch einen Hexa Viewer an, mit welchem sich die einzelnen Teile eines DICOM Datenstroms aufgezeigen lassen. Der DICOM Sante PACS Server wird in einem anderen Kapitel behandelt.

Unter folgendem Link findet man die DEMO-Software: Sante PACS Server

Konfiguration

SANTE Server:

Hier werden AE Title und Port geändert. Die IP Adresse ist nicht änderbar.

DICOM Network:

Hier werden DICOM Knoten verknüpft.

  1. Öffnen des Network Peer Setup.
  2. Mit Insert kann ein Knoten hinzugefügt werden.
  3. Hier werden der AET, Host/IP-Adresse und der Port gewählt.

Der gleiche Prozess für alle weiteren DICOM Knoten.

Auch Orthanc verfügt über die Möglichkeit, die Bilder zu betrachten. In unserem Fall wird der RPi ohne Anzeige verwendet. Mittels dem Orthanc Explorer kann via Webviever das Bild betrachtet werden. Der Orthanc Explorer lässt sich über einen beliebigen Webbrowser öffnen; http://localhost:8042/app/explorer.html.
