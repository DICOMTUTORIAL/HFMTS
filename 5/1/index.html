Vorbereitung

Überblick

In diesem Tutorial wird auf einem Raspberry Pi 4 ein DICOM Server installiert & konfiguriert. Dazu wird auch gezeigt, wie eine Modalität angebunden wird.

Wir verbinden uns von einem PC auf den RPi via SSH Verbindung. In diesem Fall wurde Putty verwendet.

Der Server im Mini-PACS ist der RPi. Die Workstation und die Modalität wird mit dem PC ausgeführt. Die Workstation kann Bilder zur Betrachtung und Bearbeitung. Die Modalität wird mit dem PC simuliert.

DICOM Viewer besitzen oft die Möglichkeit Bilder abzurufen, abzuspeichern oder zu versenden. Es wird also kein PACS Server benötigt, um Dienste auf den (Objekten) Bildern auszuführen.

Dazu wird auf dem PC auch Orthanc installiert. Danach werden DICOM Bilder auf Orthanc geladen. Somit nimmt Orthanc auf dem PC die Funktion der Modalität ein. Sie sendet erzeugte Bilder an das Archiv zur Speicherung.
